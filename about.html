About PuGo

Introduction

PuGo is a simple static site generator by Golang. It compiled markdown content to site pages with beautiful theme. No dependencies, cross platform and very fast.

Quick start

  1. Download and extract from PuGo Releases, Run pugo[.exe] server directly.
  2. Open http://localhost:9899 to visit.

Commands

Run a command when run pugo executable file:

  • pugo new create new site, post or page.
  • pugo build build static files.
  • pugo server build and serve static files.

More details in Commands.

Writing

PuGo support two kinds of content, post and page. you can create any .md file in proper directories in source directory. Read the wiki to learn the layout format and more details.

Publish

After you change your source .md files, run

$ ./pugo build

To build static files.

If you want to deploy your site, read this documentation.
